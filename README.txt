This project is a simple rule-based chatbot developed in Python as part of Project 1 of the DecodeLabs Artificial Intelligence Internship Program. 
The chatbot responds to predefined user inputs using functions, dictionaries, loops, and conditional logic. It can handle greetings, provide the 
current date and time, tell a joke, respond to help and name-related questions, and exit when requested by the user. User input is sanitized by 
converting it to lowercase and removing unnecessary spaces to improve accuracy. The main objective of this project was to understand the fundamentals 
of chatbot development, control flow, decision-making logic, and dictionary-based command handling before moving on to more advanced AI concepts.

To run the project, open a terminal in the project directory and execute:
python chatbot.py
